Disable Behaviour Node

A Disable Behaviour Node is an Instruction Graph Node that will disable a Behaviour on the given GameObject. Create a Disable Behaviour Node in the Create  Object Manipulation  Disable Behaviour menu of the Instruction Graph Window.

See the "Disable Player" node on the "MazeStart" Instruction Graph in the Maze project for an example usage.

Fields

Name Description

Target

The VariableReference to the Behaviour to disable
