Selection Node

A Message Node is an Instruction Graph Node that will tell a Message Control to shop a Message. Showing a Message will automatically activate the Message Control and will hide it when the Message is complete if AutoHide is true. If AutoHide is false then a Hide Control Node must be used to deactivate it. Create a Message Node in the Create  Interface  Message menu of the Instruction Graph Window.

See the "Level Message" node on the "MazeStart" Instruction Graph in the Maze project for an example usage.

Fields

Name Description

Control

A VariableReference to the Selection Control that should display Items

Selected Item

A VariableReference to store the Value of the selected item in

Selected Index

A VariableReference to store the index of the selected item in

Is Selection Required

Whether or not the selection can be canceled (no item is selected)

Auto Hide

Whether to deactivate the Control once the selection has been made

Items

The list of Selection Items that will appear in the selection
