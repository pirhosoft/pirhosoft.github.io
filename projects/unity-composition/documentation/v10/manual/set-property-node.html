Set Property Node

A Set Property Node is an Instruction Graph Node used to set any property or field on any Object provided that the type of the property or field can be stored in a VariableValue. Create a Set Property Node in the Create  Composition  Get Property menu of the Instruction Graph Window.

See the "Move To Open" node on the "MazeGateStart" Instruction Graph in the Maze project for an example usage.

Fields

Name Description

Target

The VariableSource to the object to set Value on

Value

The VariableValue to set the property to
