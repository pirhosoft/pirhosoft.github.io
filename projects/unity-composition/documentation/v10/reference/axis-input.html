AxisInput

PiRhoSoft.CompositionEngine.AxisInput : MonoBehaviour, IEventSystemHandler, IPointerDownHandler, IPointerUpHandler

Description

AxisInput is a MonoBehaviour that will set a value on an input axis when an object is clicked. See the manual page for usage details.

Public Fields

string AxisName

The name of the axis whose value is set when the object with this behaviour is clicked or touched.

float AxisValue

The value to set the axis to when the object with this behaviour is clicked or touched.
