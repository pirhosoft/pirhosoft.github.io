Command

PiRhoSoft.CompositionEngine.Command : ScriptableObject, ICommand

Description

A ScriptableObject type that is used to define an Expression that can be called from other Expressions as if it were a ICommand.

Public Fields

string Name

The name of the command that is used to call it from Expressions

ParameterList Parameters

The list of parameters that should be passed to the command.

Expression Expression

The Expression that is evaluated when this command is called.

Public Methods

VariableValue Evaluate(IVariableStore variables, string name, List<Operation> parameters)

Validates parameters against the types defined in Parameters and, if valid, executes Expression. The result of executing Expression is returned.
