ConditionalNode

PiRhoSoft.CompositionEngine.ConditionalNode : InstructionGraphNode

Description

An InstructionGraphNode that continues to another node based on the result of a conditional Expression.

Public Fields

InstructionGraphNode OnTrue

If Condition evaluates to true, this node will run.

InstructionGraphNode OnFalse

If Condition evaluates to false, this node will run.

Expression Condition

The expression to execute to determine which node should be run. The expression should return a bool, otherwise an error will be logged.
