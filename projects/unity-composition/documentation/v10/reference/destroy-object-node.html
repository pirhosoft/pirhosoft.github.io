DestroyObjectNode

PiRhoSoft.CompositionEngine.DestroyObjectNode : InstructionGraphNode

Description

Destroys an object.

Public Fields

InstructionGraphNode Next

The node to run after the object is destroyed.

VariableReference Target

The reference to the variable holding the object that is to be destroyed. The object can be any Object. If it is a MonoBehaviour, the owning GameObject will be destroyed.
