MappedVariableAttribute

PiRhoSoft.CompositionEngine.MappedVariableAttribute : Attribute

Description

An attribute to apply to properties and fields to expose them to the variables system using MappedVariableStore.

Constructors

MappedVariableAttribute(bool readable, bool writable)

Sets the allowed access for the property or field the attribute is applied to. If the default constructor is used instead, readable and writable are both true.

Public Fields

bool Readable

Set this to true to allow the property or field value to be retrieved when accessed through a MappedVariableStore.

bool Writable

Set this to true to allow the property or field value to be set when accessed through a MappedVariableStore.
