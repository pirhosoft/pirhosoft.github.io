PlayAnimation

PiRhoSoft.CompositionEngine.PlayAnimation : InstructionGraphNode

Description

Plays an AnimationClip on an AnimationPlayer.

Public Fields

The InstructionGraphNode to run when this node completes.

The AnimationPlayer to play Animation on.

The AnimationClip to play on AnimationPlayer.

bool WaitForCompletion

If this is true, this node will not complete until Animation has completed. Otherwise, this node will complete immediately and the InstructionGraph will continue.
