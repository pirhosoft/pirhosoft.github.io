PlaySound

PiRhoSoft.CompositionEngine.PlaySound : InstructionGraphNode

Description

Plays an AudioClip on an AudioPlayer.

Public Fields

InstructionGraphNode Next

The InstructionGraphNode to run when this node completes.

VariableReference AudioPlayer

The AudioPlayer to play Sound on.

AudioClipVariableSource Sound

The AudioClip to play on AudioPlayer.

FloatVariableSource Volume

The volume to set on AudioPlayer when playing Sound.

bool WaitForCompletion

If this is true, this node will not complete until Sound has completed. Otherwise, this node will complete immediately and the InstructionGraph will continue.
