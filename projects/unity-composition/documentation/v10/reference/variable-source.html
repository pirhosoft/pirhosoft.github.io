VariableSource

PiRhoSoft.CompositionEngine.VariableSource

Description

A wrapper type for fields that allows a variable to be set in the editor to a specific value or set as a VariableReference.

Public Fields

VariableSourceType Type

Whether this source has a value or reference. If this is set to Value the subclass (such as VariableSource<T>) will include the value field of the correct type.

VariableReference Reference

If Type is set to Reference, this holds the VariableReference where the VariableValue should be looked up.

Public Methods

void GetInputs(IList<VariableDefinition> inputs) (abstract)

If Type is set to Reference and Reference accesses the input store, subclasses will add the definition for Reference to the inputs list.
