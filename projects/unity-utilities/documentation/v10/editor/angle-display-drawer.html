AngleDisplayDrawer

PiRhoSoft.UtilityEditor.AngleDisplayDrawer : PropertyDrawer

Description

Draws an editor for a Quaternion. Use the AngleDisplayAttribute to automatically use this drawer or the static methods to draw a Quaternion manually (i.e from a drawer, control, or editor).

Static Methods

float GetHeight(GUIContent label, AngleDisplayType type)

Returns the height the drawer needs to display a Quaternion with the specified label and type.

void Draw(GUIContent label, SerializedProperty property, AngleDisplayType type)

Draws the editor for a property referencing a Quaternion field using the specified label and type with automatic layout.

void Draw(Rect position, GUIContent label, SerializedProperty property, AngleDisplayType type)

Draws the editor for a property referencing a Quaternion field using the specified label and type at the specified position and size.

Quaternion Draw(GUIContent label, Quaternion quaternion, AngleDisplayType type)

Draws the editor for a Quaternion using the specified label and type with automatic layout. Changes are reflected in the return value.

Quaternion Draw(Rect position, GUIContent label, Quaternion quaternion, AngleDisplayType type)

Draws the editor for a Quaternion using the specified label and type at the specified position and size. Changes are reflected in the return value.
